Garden of Humans (GOH) – Official Website
A modern, responsive website for Garden of Humans (GOH) built with React.
This project showcases the band’s identity, music, and live events while providing a scalable foundation for future features.

Status: In development (MVP in progress)

Live site: Coming soon (Netlify deployment)

Purpose:

This project is being built to:
Create a strong digital presence for the band
Showcase music, gigs, and visuals
Practice React component architecture
Build a portfolio-ready project with real-world features

Tech Stack:

React (Component-based architecture)
JavaScript (ES6+)
CSS / SCSS
Netlify (deployment & forms)

Key Concepts Practiced:

Component reusability
Props & data mapping
Responsive layouts (Flexbox/Grid)
Form handling
External integrations (Spotify, etc.)


Planned Features (MVP):

👥 About Section
Reusable BandMember component
Grid layout of members
Image + short bio per member
🎧 Music Section
Embedded music player
Links to platforms (Spotify, SoundCloud)
Highlight featured tracks
🎤 Gigs Section
List of upcoming events
Date, location, ticket links
Easily updatable data structure
📬 Contact Section
Contact form (Netlify Forms)
Social media links
Booking enquiries
